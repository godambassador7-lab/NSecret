# NSecret

**Unseen good. Real impact.**

A private gamified virtue RPG where progress is meaningful even when unseen.

## Philosophy

- No social sharing
- No proof required
- No external validation
- Trust in conscience
- Loss can be virtuous

## Installation

1. Install dependencies:
```bash
npm install
```

2. Install Tailwind CSS:
```bash
npm install -D tailwindcss postcss autoprefixer
```

3. Start the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Features

- **Daily Secret Acts**: Receive acts of service, restraint, discipline, courage, or mercy
- **Integrity System**: Honest reflection on whether you told anyone
- **Hidden Progression**: Stats grow invisibly through ranks
- **Narrative Evolution**: Messages evolve from encouraging to sparse
- **Sacred Loss Events**: Voluntary sacrifice of progress
- **Private Reflection**: Optional journaling that's never analyzed
- **Local-First**: All data stored in browser localStorage

## Design

White gradients flowing into warm amber gold tones. Refined typography with Cormorant Garamond and Crimson Pro serifs. Glass-morphic cards with subtle animations.

## Privacy

All data is stored locally in your browser. No analytics, no tracking, no external servers.
